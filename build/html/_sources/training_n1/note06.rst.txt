note 06
======================

第11課　目的．手段
--------------------

+ Aべく

「Aというはっきりした目的を持ってBという行為をする。」という意味の文型です。

Aの部分には、意志動詞が入ります。

意志動詞：自分でコントロールできる動詞。（行きます、食べます、勉強します、など。）

無意志動詞：コントロールできない動詞。（可能動詞、見えます、わかります、など。）

接続：動詞の辞書形　例外：する→　するべく．すべく

#. 例1: 顧客の期待に応えるべく、商品開発を進めてきました。
#. 例2: 文章を書くのが苦手なので、それを克服すべく、短い文から少しずつ書いてみようと思う。



+ AんがためにB

「Aんがために、B」は「AするためにBする。」という意味で「Aの目的を達成する」という強い意志を表す文型です。Aには意志動詞が来て、Bには自分が意思をもってする行為が入ります。

接続：動詞のない形。

例外：3グループの「します」は「しんがために」ではなく、「せんがために」という形になります！

後件の接続について

「〜せんがため」「〜せんがために」は同様に使えます。名詞につなげる場合は「〜せんがための＜名詞＞」となります。



#. 例1：日本で働く夢を実現させんがために、ベースに勤めることにした。
#. 例2: 第一希望の難関大学に合格するという目標を達成せんがために、日々努力してきた。



+ AをもってB

「AをもってB」は「Aで/Aを使って/Aによって」という意味の文型です。AにはBをするための手段が入ります。名詞に接続します。

✓「Xをもってすれば」という慣用句的な言い方もあります。これはXを高く評価するときに使います。Xには「能力、実力、才能」などの言葉が入ります。

✓　逆接は「Xをもってしても」となります。

✓「身をもって」は慣用句的な言葉で「自分自身で・自分の経験として」という意味です。

✓「何をもって」は意味がとらえにくいですが、「どういう基準で、どういう方法で」という意味と説明します。

✓　硬い言い方です。日常的・具体的な道具などには使いません。

×はしをもって、ご飯を食べます。→◯はしで、ご飯を食べます。

#. 例1: 審査の結果は、 2週間以内に書面をもってお知らせします。
#. 例2: 新しい社長は優れたリーダリップを持って古い組織を改革した。
#. 例3: 何を持って人を評価するかは、未だにコンセンサスが得られていない難しい問題だ。



第12課　原因．理由
------------------------

+  AばこそB

「AばこそB」は「Aだから、Bになる/Aだからあえて、Bをする/BになるのはAの場合だけだ。」という意味の文型です。普通は原因として考えにくいことをあえて理由として強調したい時に使う文型です。強調構文の形になって、「BなのはAこそだ」という形になることもあります。「ば」で分かると思いますが、条件形と接続します。



#. 例1: 娘の将来を思えばこそ、多額の借金がある男との結婚には賛成できない。
#. 例2: (上司が部下に)　「心身ともに健康であればこそ、大きな仕事に挑戦できるのだ。まずは健康に留意しなさい。」
#. 例3: 君のことを思えばこそ、すぐにはヒントを出さないんだよ。まずは自分の頭で考えなさい。



+ AとあってB

「AとあってB」は「通常とは違い、Aという状況（事情）なので、当然Bだ。」という意味の文型です。普通形に接続しますが「な形容詞」と「名詞」の場合は「だ」が落ちる場合が多いです。

✓　Aには普通ではない特別な言葉を表す言葉が入ります。

✓　BにはAの状況から考えられる当然の結果をいいます。

✓　自分のことを言う時にはあまり使いません。



#. 例1: 5年ぶりの再会とあって、時間が経つのも忘れてお互いの近況を報告しあった。
#. 例2: ここは路線が複数通っていて便利とあって、家賃が非常に高いです。
#. 例3: 昨日は憧れていた彼女との初デートとあって、僕はなかなか眠ることができなかった。



+ AではあるまいしB

「AではあるまいしB」は「Aならそのようなことがあるかもしれないけど、実際はAじゃないのだから、B」という意味の文型です。

後件Bには非難・忠告・判断などがきます。今の状態を否定するような意味の文が入ります。

接続：「では」の部分は、会話では「じゃ」になることもあります。

「普通体＋わけ/の/ん＋では（じゃ）あるまいし」という形もあります。



#. 例1: 一生会えないわけではあるまいし、そんなに泣かなくてもいいでしょう。
#. 例2: 新入社員ではあるまいし、基本的なことはいちいち説明する必要ないと思います。



+ Aてまえ、B

「てまえ」は漢字で「手前」と書きます。「人が見ている前。他人に対する自分の立場」という意味があります。

「Aてまえ、B」は「Aしてしまったから、Bしなければならない。」「Aという立場・人」を意識するから、Bしないと自分の評価が下がってしまう」という意味の文型です。

✓　後件Bには「なければならない」「わけにはいかない」「ざるをえない」などの行動を強制（制限）するような文がきます。

✓ 「他の人の気持ちや社会的な評価を考えて、（気が進まないけど）Bする」という時に使います。

接続：動詞の辞書形/た形/動詞のて形+いる/名詞＋の

#. 例1: 夫がCanonに勤めている手前、他社のカメラが欲しいなんて言えない。
#. 例2: 夫に言いたいことは山ほどあるが、子どもが見ている手前、けんかするわけにはいかない。



+ AゆえにB

「Aゆえ（に）B」は「AだからB。」という意味の文型です。

✓　古い書き言葉で、手紙やあらたまった場面で使います。

✓　形容詞は名詞化して＜ーさゆえに＞という形になることも多いです。

✓　名詞に接続する場合「ーゆえの＜名詞＞」となります。

 **普通形としましたが、以下は変則的なので注意します** 。

::

	•	な形容詞：な形容詞だ→な/である
	•	名詞：名詞だ→な/である（「な」がつかないこともあります。）


また「名詞修飾型」という型を教えているのなら、「名詞修飾型＋ゆえに」と一言で説明してもいいです。学生が理解しやすい説明を選びます。

参考：＜名詞修飾型＞

動詞：食べる｜食べない｜食べた｜食べなかった

い形容詞：おいしい｜おいしくない｜おいしかった｜おいしくなかった

な形容詞：親切な｜親切じゃ（では）ない｜親切だった｜親切じゃ（では）なかった

名詞：学生の｜学生じゃ（では）ない｜学生だった｜親切じゃ（では）なかった


#. 例1: 急な話ゆえ、君が驚くのも当然のことだ。
#. 例2: 慣れないことゆえ、数々の不手際、どうぞお許しください。
#. 例3: 未成年も法で裁かれるべきだ。若さ故の過ちにも罰を与えなければならない。
#. 例4: 若さゆえの過ちというものは、誰しも認めたくないものだ。
