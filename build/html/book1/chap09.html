句型¶

一、

（体言）は（体言）です。 / ……是……。
（体言）も（体言）です。/ ……也是……。

  • これは鍵です。

  • それも鍵です。

  • 私は学生です。

  • あの人も学生です。

二、

（体言）は（体言）ではありません。 / ……不是……。
（体言）も（体言）ではありません。 / ……也不是……。

  • これは鞄ではありません。

  • それも鞄ではありません。

  • 私は先生ではありません。

  • あの人も先生ではありません 。

三、

（体言）は（体言）ですか。 / ……是……吗?
はい／是的。
いいえ／不是。

  • これは鍵ですか？はい、それは鍵です。

  • それも鍵ですか？はい、これも鍵です。

  • あれも電話ですか？いいえ、あれは電話ありません。

四、

（体言）は（疑问词）ですか。 / ……是（谁、什么、哪里）吗?

  • これは何ですか？

  • 花瓶はどれですか？

  • あの人は誰ですか？

五、

（体言）は（体言）ですか、（体言）ですか。/……是……,还是……?

  • これは本箱ですか？ダンスですか？

  • 田中さんは先生ですか？学生ですか？

语法¶

一、判断句¶

判断事物是什么或不是什么的句子叫判断句，特征是由体言加判断助动词作谓语。

二、判断助动词「です」¶

接在体言或某些助词、副词的后面表示断定。还有增添陈述的作用，从而构成判断句。有词尾变化，肯定式为「です」，否定式为「ではありません」。

两个判断句连成一个时，使用「です」的连用形「で」表示中顿（中止法）例如：

  • 私は学生で、あなたはせんせいです。

三、助词「は」「も」「か」「の」¶

助词是不能用来单独表达具体概念的附属词，接在其他词的后面，确定词语词与词之间的关系及词在句中的地位，或者给词增添某种意义。助词分为六类：格助词，接续助词，提示助词，副助词，并列助词和语气助词。

  1. 提示助词「は」

    提示助词接在体言后面，把该体言作为主题提出来加以提问，或说明可以提示主语，也可以提示宾语等。

  2. 格助词「も」

    接在体言后面，说明与另一个事物是相同的，相当于汉语中的“也”。

  3. 格助词「の」

    格助词接在体言后面，构成定语，表示领属关系，相当于汉语中的“的”。

  4. 语气助词「か」

    用于句末，表示疑问的语气，相当于汉语中“吗”、“嗯”。

四、名词¶

表示人或事物的名称的词叫做名词，名词是没有词尾变化的，独立词属于体言。名词又可分为普通名词，专有名词，时间名词和形式名词等。除形式名词不可单独使用外，其余都可以单独使用。

五、代词¶

只带人或事物的单词叫代词，代词可以分为指示代词和人称代词，每种都有近称中称远称和不定称的区别。

六、连体词¶

连体词不能单独使用，只能加在体言前面修饰体言。本课出现的是表示指示的连体词。

七、感叹词「はい」「いいえ」¶

感叹词是无词尾变化的独立词，表示感叹、招呼、应答等意义。

八、接续词「それでは」¶

接续词是无词尾变化的独立词，他对前后的词或句子起承上启下的作用。

「それでは」是顺接意义的接续词，相当于汉中的“那么”。

九、接尾词「さん」「くん」¶

「さん」接在比自己地位、辈份高或平辈的对方和第三者姓名或职务名称之后，表示尊敬与亲切之意。「くん」接在比自己地位、辈份低的或平辈的第三方姓名或职务之后，表示亲切之意。
