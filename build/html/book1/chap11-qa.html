答案¶

口头练习¶

三、

  1. いいえ、あまり静かではありません。

  2. いいえ、そんなに重くないです。

  3. はい、大変高いです。

  4. いいえ、あまり有名ではありません。

  5. いいえ、そんなに賑やかではありません。

  6. はい、大変きれいです。

  7. はい、いつも静かです。

四、

  1. 私の学校はデパートに近いです。

  2. あの高い建物は私の教学館です。

  3. あの教室はたいへん明るいです。あの教室はあまり明るくないです。

  4. あの新しい小説はたいへん面白いです。

  5. この町はとてもきれいです。交通も便利です。

  6. 李さんは真面目な学生です。

笔答练习¶

一、

  1. どんな

  2. 何の

  3. どう

二、

  1. 私たちの教学館はあまり高くありません。

  2. 王さんの学校は駅にもデパートにもちかいです。たいへん便利です。

  3. 日本は桜で有名の国です。山も水もたいへんきれいです。

  4. 田中さんの会社はどうなところにありますか。都心に近い、たいへん賑やかな所にあります。

  5. 図書館の中にはいつも静かです。

  6. あの赤いリンゴはあまりおいしくないです。
