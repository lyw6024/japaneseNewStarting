    答案¶

    口头练习¶

    一、

    1. 私たちの宿舎は部屋が狭くて、窓がちいさいです。けれども、そんなに暗くありません。

    2. 日本語は英語よりむずかしいです。英語は日本語ほど難しくないです。

    3. 富士山は日本で一番高い山です。。

    4. 中国と日本と、どちらが人口が多いですか。日本より、中国の方が人口が多いです。

    5. このカメラは小さくてべんりです。

    笔答练习¶

    一、

    1. は　く　て

    2. と　と　が

    3. は　よ　り　は　ほ　ど

    4. で　は

    5. と　と　が　も

    6. が　か　ほ　ど

    二、

    1. 暑い　暑い　暑くて　暑く

    2. 丈夫　丈夫な　丈夫で　丈夫

    三、

    1. あなたたちの教室と図書館と、どちらがしずかですか。

    2. どちらも静かです。けれども、教室は図書館ほど静かではありません。

    3. 象は鼻が長くて、耳が大きくて、目が小さいです。

    4. 李さんと王さんと、誰が体が丈夫ですか。

    5. 李さんより王さんの方が体が丈夫です。

    6. 上海は中国で一番人口が多いです。けれども、東京ほど人口が多くではありません。

    7. この高くて新しい建物は教学館で、その低くで古い建物は図書館です。
