句型
-------
一、 ::
	
	 （体言）に（は）（体言）があります。 /在……（地方）有……（东西）。
	 （体言）に（体言）はありません。 /在……没有……。


- ここに本があります。
- ここに本はありません。
- 机の上に万年筆があります。
- 机の上に万年筆はありません。

二、 ::
	
	（体言）もありますか。/也有……（东西）吗?

- 机の上にま万年筆もありますか？
- はい、万年筆もあります。
- いいえ、万年筆はありません。

三、 ::
	
	（体言）は（体言）にあるます。
	（体言）は（体言）にありません。


- 私の家は銀行のそばにあります。
- 私の家は銀行のそばにありません。
- 本は机の中にあります。
- 本は机の中にありません。

四、 ::
	
	（体言）は（疑问词どこ）にありますか。

- あなたの家はどこにありますか？私の家は銀行のそばにあります。
- 鉛筆はどこにありますか？鉛筆は机の上にあります。

语法
------------

一、存在句
^^^^^^^^^^^^^
叙述事物是否存在的句子。特征是有「ある」作谓语。


二、场所指示代词
^^^^^^^^^^^^^^^^^^^^^^^^^^^^
近称ここ、中称そこ、远称あそこ、不定称どこ

三、格助词「が」「に」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^
格助词主要是接在体言后面，决定体言在句中的地位。

1. 格助词「が」

   - ここに新聞があります。
   - そこに何がありますか？

   区别「は」与「が」：当以疑问句作主语时，主语后边用「が」，回答时也用「が」。当以疑问词作谓语时，主语后用「は」，回答也用「は」。
   有时句子没有疑问词，但内含疑问之意，为突出主语，此时也用「が」。

   - どれがあなたの本ですか？それが私の本です。
   - あなたの本はどれですか？私の本はそれです。
   - それはあなたの本ですか？いいえ、これは私の本ではありません。

2. 格助词「に」

   - 机の上に本があります。
   - 本は机の上にあります。




四、并列助词「と」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^
用于体言和体言之间，表示并列，相当于汉语“和”的意思。

- それは鉛筆と本です。
- ここには、タバコと灰皿がありま。


五、「何か」「何が」「何も」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

「何か」的か是副助词，在疑问词或不定代词后，表示不定。
「何が」提问的是存在东西的名称，回答时只需要答出是什么。
「何も」与否定连用，表示全面否定。

- 机の中に何がありますか？はい、ありますいいえありません。
- 机の上に何がありますか？机の上に本があります。
- 机の下には何もありません。


六、形容词
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

表示事物性质、状态的词叫形容词。在活用时，没有变化的部分是词干，有变化的部分是词尾。
形容词以「い」结尾。形容词的连体形放在体言前边，连体形的词尾仍是「い」。

- これは大きい工場です。
- それは新しい鞄です。


七、提示助词「も」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^

以「…も…も」的形式并提两个或两个以上的人或事物，相当于“……和……都……”。


- ここには大きい鞄も小さい鞄もあります。
- 庭に白い花も赤い花もあります。
