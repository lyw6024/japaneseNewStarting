第八课 新聞を読んだりします
==========================================


课文
---------------
田中さんは毎朝新聞を読んだり、テレビを見たりします。

田中さんは駅の売店でスポーツ新聞を読んだり、週刊誌を書いてしたりします。たくさんの人が電車の中で新聞を読んだりしかしを読んだりします。

日本人は毎日新聞を家で読んだり、電車の中で読んだりします。新聞のかいにテレビを見たり、ラジオを聴いたりしてニュースを知ります。

日本の新聞は一般紙と専門紙があります。一般紙には朝刊と夕刊があります。専門家にはたくさんの種類があります。スポーツの新聞だったり経済の新聞だったりです。

新聞社によって同じニュースでも見たしが大きかったり小さかったりします。ケチキン。景色の着方も新聞社によって違います。人々は自分の仕事や趣味によって新聞を選びます。

会话
--------------


張：日本の新聞の種類が多いですね。

田中：ええ、多いですよ。日本人は毎日家で読んだり、電車の中で読んだりします。

張：そうですね。朝の電車の中でほとんどの人が新聞を読んだり、週刊誌を読んだり知っていますね。

田中：張さんは1日何種類の新聞を読みますか？

張：1種類しか読みません。朝刊と夕刊だけです。

田中：私は3種類です。一般紙とスポーツ新聞と経済新聞を読みます。

張：すごいですね。

田中：新聞のほかに夜家でテレビを見たりラジオを聴いたりします。

張：色々な方法にニュースを知ることができますね。

句型
------------

一、 ::

    （形容词连用形かっ、体言／形容动词连用形だっ）たりします・です。／时而……，时而……。

- 見出しが大きかったり小さかったりです。
- 天気が暖かかったり寒かったりします。
- スポーツの専門だったり、経済だったりします。
- 時間によって、しずかだったり、にぎやかだったりします。

二、 ::

    (体言)によって/根据，由于

- 記事の書き方も新聞社によってちがいます。
- 自分の趣味によって新聞を選びます。
- 季節によって服をかえます。


三、 ::

    (动词连体形)ことができます。/ 能够……，会……。


- いろいろな方法でニュースを知ることができます。
- 私は日本語を話すことができます。
- 日本語で文章を書くことができません。

语法
------------

一、接续助词「たり（だり）」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

「たり」接在用言和助动词的连用形之后，动词拨音便及ぐ尾イ音便时变为「だり」，经常多个并列使用，也可以作为连用修饰语修饰后边的用言，有时也可看作一个体言

1. 表示列举

列举同类或相反的动作、作用，或表示动作、状态的交替发生或反复。


- 子供が飛んだり、跳ねたりします。
- 人々が出たり、入ったりしてとても賑やかです。
- このごろは寒かったり、暖かかったりします。
- 山を越えたり、川を渡ったり、できるだけ直線コースを取ります。
- 出発する前に飲んだり、食ったり、充分腹をこしらえます。
- 着物も綺麗だったり污かったり、さまざまです。
- 彼の話は本当でしたり嘘でしたり、信じることができません。
- この二つは似たり寄ったりですね。
- 田中さんは来たり来なかったり、決まりがません。
- ホテルでは、部屋が立派だったり、部屋に風呂場が有ったりすると、値段はずっと高くなります。
- 動物園の熊はいつもおりの中で行ったり来たりします。
- 人間はみんな喜んだり、悲しんだりして、一生を過ごします。



2. 表示概括

举出一两个动作、作用，概括同类的其它，这种场合可以只使用一个「たり」，并且最后「たり」可以附加「など」

- 日曜日には洗濯をしたり、買い物をしたりします。
- 壁などに落書したりなどしてはいけません。
- あの人はいつも噓をついたりします。


二、接续助词「て」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

接在形容词、动词、助动词的连用形以及五段动词音便形之后。

1. 表示并列中止

- 昼間は明るくて、夜は暗いです。
- 日は東から出て、西にはいります。
- 兄は工場へ行って、妹は学校へ行きます。

2. 表示方法、手段

- 先生は立って講義をします。
- みんなは手をたたいて歓迎します。
- テレビを見たり、ラジオを聞いたりしてニュースを知ります。

3. 表示原因

- 部屋が暗くて新聞を読むことができません。
- この靴は小さくて履くことができません。
- 午後会議があって空港へ行くことができません。


4. 表示顺序（与「てから」表示顺序的用法相近）

- 私はに行って雑誌を読みます。
- 朝起きるとはをみがいて顔を洗います。
- みんなは入場券を買って映画館に入ります。


三、形容词连用形
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

形容词连用形有两种形态

1. 「かっ」形，后续「た」「たり」

- 昨日は寒かったです。
- 天気は寒かったり、暑かったりします。

2. 「く」形后接「ない」「ありません」「なる」，或用来修饰用言

- 寒くないです。
- 寒くありません。
- 寒くなります。
- 朝早く起きます。
- 夜遅く寝ます。

四、形容动词连用形
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

形容动词的连用形主要有三种形态：

1. 「だ」形，后续「た」「たり」

- 二、三年前ここはきれいだった。
- 静かだったり、にぎやかだったりです。

2. 「で」形，主要表示并列中止

- あの町は静かできれいです。

3. 「に」形，后续用言，构成连用修饰语，当后续动词「なる」时，表示状态的转变

- 教室をきれいに掃除します。
- 体が丈夫になります。



五、「には」的用法
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

「には」用于提示存在的场所，后续谓语为存在动词。

- 日本の新聞には一般紙と専門紙があります。
- ペキンには大きい建物がたくさんアルマス。
- 教室の中には学生が大勢います。


六、副助词「だけ」
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

接在体言后边，表示限定。相当于汉语的“只、仅与「しか……ない」的意义类似。

- 一般紙は朝刊と夕刊だけです。
- 新聞は一種類だけ読みます。＝新聞を１種類しか読みません。
- 先生は机は一つだけです。＝先生の机は一つしかありません。
